## About This Document

*Learning Through Play* is a book-length design analysis of an educational mathematics games platform, written from two expert perspectives:

- **Dr. Ruth Cohn** — Instructional designer and educational researcher
- **Farai Dube** — Senior game developer

The analysis applies established frameworks from educational game design research — including the MDA Framework, the RETAIN Model, Flow Theory, Cognitive Load Theory, and Self-Determination Theory — to ten games on the platform, producing specific, actionable design recommendations.

Estimated total length: approximately 24,000 words.
